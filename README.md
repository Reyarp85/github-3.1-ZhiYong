# Basic Git Commands

## 📝 Setting Up Git

- **Configure user name:**  
  🖊️ `git config --global user.name "Your Name"`

- **Configure email:**  
  📧 `git config --global user.email "your.email@example.com"`

- **Check configuration:**  
  🔍 `git config --list`

## 📁 Working with Repositories

- **Clone a repository:**  
  📥 `git clone https://github.com/username/repository.git`
